# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-05-09

### Added
- Semantic retrieval foundation with chunk schema, embedding abstraction, and local vector index.
- Semantic upgrades for `mnemo_similar`, `mnemo_knowledge`, and `mnemo_search_api`.
- Architecture classification, task-aware context retrieval, and hierarchical repo map output.
- Local distribution assets: PyPI release workflow, binary prototype scripts, and VS Code extension MVP scaffold.

### Changed
- Parser coverage now includes improved JavaScript/TypeScript extraction and Go extraction.
- MCP server version now reads package version constant.
